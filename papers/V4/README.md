# papers/V4

This frozen package contains the current best available manuscript, response-to-reviewers draft, cover letter draft, figures, and tables for the cardiac RFA reduced electro-thermal modeling study.

## Contents
- manuscript/: submission-ready manuscript in DOCX, PDF, and Markdown
- response/: response-to-reviewers draft in DOCX, PDF, and Markdown
- cover_letter/: cover letter draft in DOCX, PDF, and Markdown
- figures/: main and supplementary figures in PDF, TIFF, and PNG preview formats
- tables/: CSV exports for protocol definitions, solver/UQ settings, benchmark points, and source data
- notes/: manifests and version notes

## Note
This refreshed package was generated to replace expired download links. It preserves the current manuscript/reviewer materials available in the workspace and organizes them into the agreed `papers/V4/` structure.
