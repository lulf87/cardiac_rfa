# papers/V5

This folder contains the frozen V5 paper package for the `cardiac_rfa` project.

## Contents
- `manuscript/`: submission-ready manuscript files (`.docx`, `.pdf`, `.md`)
- `response/`: response-to-reviewers files (`.docx`, `.pdf`, `.md`)
- `cover_letter/`: cover letter files (`.docx`, `.pdf`, `.md`)
- `figures/`: frozen main and supplementary figures, separated into `pdf`, `tiff`, and `png_preview`
- `tables/`: frozen CSV table exports and source CSV inputs
- `notes/`: version notes and file manifests

## Placement
Extract this package at the repository root so the final path is:

`~/Projects/cardiac_rfa/papers/V5/`

## Scope of V5
V5 carries forward the latest frozen manuscript, figures, tables, response letter, and cover letter into the agreed paper-package structure. It is intended as the next clean handoff version after V4.
