# Cardiac RFA paper package V6

This directory contains the frozen V6 paper package prepared after the major-revision technical cleanup.

## Contents
- `manuscript/`: main manuscript in DOCX, PDF, and Markdown
- `response/`: response-to-reviewers package
- `cover_letter/`: cover-letter package
- `figures/`: main and supplementary figures in PDF, TIFF, and PNG preview formats
- `tables/`: main tables and source CSV files
- `notes/`: version notes, checklist, and package manifest
