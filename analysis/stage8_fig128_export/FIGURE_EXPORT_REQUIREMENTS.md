
# Figure export requirements (practical checklist)

Use this checklist before submission.

## Recommended master outputs

For every main figure, save:

- `figureX_*.pdf`  → preferred submission master for schematic / line / scatter / mixed vector figures
- `figureX_*.png`  → high-resolution preview copy

## Recommended local policy

- Keep text at a size that remains readable when reduced to the final journal width.
- Keep all figures as separate files.
- Number figure files in manuscript order.
- Keep captions in the manuscript text, not burned into the artwork except for panel labels and essential in-figure annotations.

## Practical defaults in this project

- PNG preview export: 600 dpi
- PDF export: vector master
- Panel labels: `(a)`, `(b)`, ...
- Line/scatter/schematic figures: prefer PDF for submission
- Heatmaps / field maps: keep both PDF and PNG

## Single-column / double-column check

Before submission, visually check each figure at:

- single-column width
- double-column width

If any text becomes hard to read, increase font size and re-export.
