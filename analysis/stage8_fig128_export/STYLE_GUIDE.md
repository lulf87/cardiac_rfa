
# Figure and export guide for stage8

## Goal

This patch fills the three missing paper figures:

- **Figure 1**: schematic / workflow
- **Figure 2**: representative spatial fields
- **Figure 8**: literature-derived benchmark

## Export policy

Every figure is exported as:

- `PDF` for journal submission / vector archiving
- `PNG` for preview and discussion

Default PNG export is `600 dpi`.

## Practical submission notes

Use the PDF version whenever possible for:

- schematics
- line plots
- scatter plots
- mixed vector figures

Keep PNG as a preview or fallback.

## Figure roles

### Figure 1
Use as the first main figure in the manuscript.

### Figure 2
Use to show the spatial interpretation of lesion formation under a representative condition.

### Figure 8
Use only after filling the benchmark template with literature-derived data points.
